An intelligent solution that is easy to install, practical and affordable and prevents deaths due to heat-stroke in children and pets
**Key design considerations:**

It must detect both heat and motion accurately.
It should send automatic alerts and escalate if needed.
It should take measures to reduce casualties by lowering the temperature.
It should operate from the vehicle’s battery with backup power.
It should be affordable and easily installable in old and new vehicles.
